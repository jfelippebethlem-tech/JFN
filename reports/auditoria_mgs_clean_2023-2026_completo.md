## 4. Como Obter Dados Reais de OBs (Ordens Bancárias Pagas)

Os dados reais de OBs (pagamentos efetivos, não empenhos) requerem acesso ao SIAFE,
que exige IP da rede governamental. Execute na máquina com acesso ao SIAFE:

```bash
# 1. Abrir Chrome com depuração remota
chrome --remote-debugging-port=9222 --user-data-dir="C:/JFN/profile"

# 2. (Opcional) Abrir TFE para empenhos:
python _SANDBOX/tfe_fornecedor.py 19.088.605/0001-04 2023
python _SANDBOX/tfe_fornecedor.py 19.088.605/0001-04 2024

# 3. Para OBs reais do SIAFE (2023-2026):
python _SANDBOX/coletar_obs_siafe_completo.py 2023 2024 2025 2026

# 4. Gerar PDF com tabelas por órgão/mês/OB individual:
python _SANDBOX/gerar_relatorio_obs_pdf.py
```
