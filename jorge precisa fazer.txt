JFN COMPLIANCE AGENT — PASSOS QUE JORGE PRECISA FAZER
======================================================
(criado em 01/06/2026)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. CONFIGURAR ACESSO REMOTO AO COMPUTADOR PELO CELULAR
   (fazer 1 vez só, no computador, antes de sair de casa)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

a) No computador, abra o Chrome e acesse:
   https://remotedesktop.google.com/access

b) Clique em "Configurar acesso remoto"

c) Baixe e instale o aplicativo que ele pedir

d) Dê um nome ao computador (ex: "PC Casa")

e) Crie um PIN de 6+ dígitos para acesso seguro

f) No celular, instale o app:
   "Chrome Remote Desktop" (Google — gratuito)

g) Abra o app, faça login com a mesma conta Google do PC

h) Pronto — pelo celular você vê e controla a tela do PC


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. CONFIGURAR INICIALIZAÇÃO AUTOMÁTICA DO AGENTE
   (fazer 1 vez só, assim o agente sobe quando o PC liga)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

a) No computador, aperte  Win + R

b) Digite:  shell:startup   e pressione Enter

c) Uma pasta vai abrir

d) Dentro dela, crie um atalho para o arquivo:
   C:\CAMINHO\DO\JFN\atualizar_e_iniciar.bat

   (Clique direito na pasta > Novo > Atalho
    No campo "local", cole o caminho completo do .bat)

e) Pronto — a partir de agora, toda vez que o PC ligar
   o agente vai automaticamente:
   - Puxar atualizações do código (git pull)
   - Subir o scheduler + bot Telegram


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. INICIAR MANUALMENTE (quando o PC já está ligado)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Abra o CMD na pasta do JFN e rode:

   git pull origin claude/rj-finance-agent-BYlhJ
   iniciar.bat

OU simplesmente dê duplo clique em:
   atualizar_e_iniciar.bat


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. CONTROLAR O AGENTE PELO CELULAR (via Telegram)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Com o iniciar.bat rodando no PC, abra o Telegram
e mande mensagem para o bot JFN:

   /ajuda    → lista todos os comandos
   /status   → mostra se o sistema está rodando e quantas OBs foram coletadas
   /agora    → dispara uma coleta completa agora mesmo
   /obs      → lista as últimas OBs coletadas com favorecido e valor
   /relatorio → envia o PDF do dia no chat

O agente manda automaticamente um resumo todo dia às 08:00.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. REQUISITOS — O QUE PRECISA ESTAR LIGADO/ABERTO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Para coleta completa (SIAFE2 + DOERJ):
  ✅ Computador ligado
  ✅ Chrome aberto na porta 9222:
     chrome.exe --remote-debugging-port=9222
  ✅ SIAFE2 logado no Chrome (siafe2.fazenda.rj.gov.br)
  ✅ iniciar.bat rodando (ou atualizar_e_iniciar.bat)

Para só o bot Telegram responder:
  ✅ Computador ligado
  ✅ iniciar.bat rodando
  (Chrome pode estar fechado — coleta será pulada)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6. SE ALGO DER ERRADO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Mande /status pelo Telegram para ver o que está acontecendo.

Se o bot não responder, o agente caiu — use o Chrome Remote Desktop
para ver a tela do PC e reiniciar o atualizar_e_iniciar.bat.

Diagnóstico completo:
   iniciar.bat --diag
(lê as páginas reais do SIAFE2 e DOERJ e envia para o repositório)
